NPM COMMANDS
=============================================================

BUILD COMMANDS:
  npm run build              Create distribution ZIP
  npm run release            Build with confirmation
  npm run clean              Remove old ZIPs

VERSION MANAGEMENT:
  npm run version            Show current version
  npm run bump:patch         Increment patch (1.0.0 → 1.0.1)
  npm run bump:minor         Increment minor (1.0.0 → 1.1.0)
  npm run bump:major         Increment major (1.0.0 → 2.0.0)

=============================================================
QUICK START
=============================================================

1. INSTALL DEPENDENCIES:
   npm install

2. CREATE DISTRIBUTION:
   npm run build

3. OUTPUT:
   dist/cf7-custom-validation-messages-v1.0.0.zip

4. INSTALL IN WORDPRESS:
   Copy to wp-content/plugins/
   Activate plugin
   Edit CF7 form → Custom Messages tab
